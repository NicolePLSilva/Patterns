Padrão Observer:
é um padrão do tipo comportamental
com foco na comunicação entre objetos;

-Usado quando uma classe precisa avisar 
outras classes sobre uma mudança de estado;

-Essas outras classes então "observam" a classe de interesse
até que ocorra a mudança de estado;

-Para que não haja problema de perfomace as classes
que observam apenas esperam que a classe de interesse
os notifique de uma mudança;

-A classe de interesse é chamada de: Subject/Sujeito
A classe interessada é chamada de: Observer/Observador
